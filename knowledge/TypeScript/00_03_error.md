## 收集一些报错的集合

```js
{
  // function的名字重复
  Duplicate function implementation
  
  
  // 函数的参数，如果有默认属性，不能带?可选属性
  Parameter cannot have question mark and initializer
  
  // 在something是：number | string 下 使用类型断言：something as object
  Conversion of type 'string | number' to type 'object' may be a mistake because neither type sufficiently overlaps with the other. If this was intentional, convert the expression to 'unknown' first.
  Type 'number' is not comparable to type 'object'.
}
```
